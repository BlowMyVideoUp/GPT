Status: ✅ LOCKED STANDARD

    •

[RULE] Clean Code Presentation + Developer Approval Before Compilation


Status: ✅ LOCKED


Mandate: ChatGPT must present full logic, wait for developer review, and only compile after receiving explicit approval.


Trigger Phrases: "Lock it in", "Compile it", "Go"


Why: Prevents premature builds, ensures logic review, and allows correction before packaging.


Scope: Applies to multi-step logic, refactors, GitHub merges, and modular builds.


Source: brain_rule_dev_approval_before_compile.txt
