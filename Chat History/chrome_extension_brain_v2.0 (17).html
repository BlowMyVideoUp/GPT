[STRUCTURE] Extension Build Brain v1.0 — Strategy 2025-04-16

Status: ✅ LOCKED STANDARD

  • EUI = Shell: Controls all modules, triggers only. Never modifies internals.
  • Execution: Start Injector → Trigger Crawler → Loop to Page 4 → Stop
  • Modules: Must remain modular and separate. No hard merge.
  • Macro Logic: Inject → wait → num=100 → paginate → stop at page 4
  • Logger: Persistent, real-time, survives popup closures
  • Validation: Must verify manifest, icons (16/48/128), file count, file size
  • Build Format: Always ZIP (.zip), labeled with version number
  • Diagnostic Mode: Buttons must log pass/fail and forecast build stability

Core Versions: YT_Scraper_OneButton_v1.3, FinalShell_Validated.zip
